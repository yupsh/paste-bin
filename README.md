# yup-paste

```
NAME:
   paste - merge lines of files

USAGE:
   paste [OPTIONS] [FILE...]

      Write lines consisting of the sequentially corresponding lines from
      each FILE, separated by TABs, to standard output.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --delimiters value, -d value  use characters from LIST instead of TABs
   --serial, -s                  paste one file at a time instead of in parallel (default: false)
   --zero-terminated, -z         line delimiter is NUL, not newline (default: false)
   --help, -h                    show help
```
